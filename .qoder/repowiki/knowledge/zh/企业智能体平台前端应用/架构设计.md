- **技术栈与构建**：采用 Vite + TypeScript + React 18 架构，使用 Ant Design 5.x 作为 UI 组件库，Zustand 进行全局状态管理。
- **路由与布局**：通过 `react-router-dom` 实现客户端路由，`AppLayout` 组件提供包含侧边栏导航和顶部用户信息的受保护布局，并通过 `ProtectedRoute` 包装器实现基于 Token 的访问控制。
- **数据交互**：封装 Axios 实例（`src/services/api.ts`）统一处理 API 请求，配置了自动注入 Bearer Token 的请求拦截器以及处理 401 未授权跳转的响应拦截器。
- **状态管理**：使用 Zustand 创建 `authStore`（`src/store/auth.ts`），负责持久化存储用户认证信息（Token 及用户详情）至 localStorage，并提供登录/登出逻辑。
